# 🍬 糖果大乱斗

一个有趣的休闲对战游戏！

## 🎮 游戏特点

- 5个完整的游戏页面（启动页、主页、角色选择、对战、结算）
- 4个可爱的角色选择
- 实时对战系统
- 精美的动画效果
- 响应式设计，支持移动设备
- 纯前端实现，无需后台

## 🚀 一键部署

### 方式一：Vercel（推荐，最简单）

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/你的用户名/你的仓库名)

### 方式二：GitHub Pages

1. 将代码推送到 GitHub
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，Folder 选择 `/ (root)`
4. 点击 Save，几分钟后就可以访问了！

### 方式三：Netlify

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/你的用户名/你的仓库名)

## 🎯 游戏玩法

1. 点击"开始游玩"进入主页
2. 选择"快速加入游戏"
3. 挑选你喜欢的角色
4. 使用攻击、防御和特殊技能与AI对战
5. 获得胜利和奖励！

## 📱 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- 无需任何框架或后台
- 响应式设计
