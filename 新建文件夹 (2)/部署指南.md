# 🚀 超级简单的部署指南（3种方式）

## 方式一：Netlify Drop（最简单！不需要 Git）⭐⭐⭐

1. 访问：https://app.netlify.com/drop
2. 把整个文件夹拖进去！
3. 等待几秒钟，就会给你一个网址！
4. 完成！🎉

---

## 方式二：Vercel（需要 GitHub）

1. 先在 GitHub 创建新仓库
2. 把代码推送到 GitHub
3. 访问：https://vercel.com/new
4. 导入你的仓库
5. 点击 Deploy，30秒后就有网址了！

---

## 方式三：GitHub Pages（完全免费）

1. 在 GitHub 创建仓库并上传代码
2. 进入仓库 → Settings → Pages
3. 选 main 分支，保存
4. 等2分钟，网址就生成了！

---

## 📁 需要上传的文件

确保这些文件都在文件夹里：

- ✅ `index.html`（游戏主文件）
- ✅ `README.md`（说明文档）
- ✅ `vercel.json`（Vercel配置）
- ✅ `.gitignore`（Git配置）
